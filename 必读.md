# Papyrus 必读

## 当前版本

- 版本：`1.0.0`
- 桌面技术栈：Tauri 2 + React 19 + Vite + TypeScript + Tailwind CSS 4 + Zustand + Tiptap
- 更新端点：`https://scallion.uno/api/papyrus/update`
- 最新下载：`https://scallion.uno/api/papyrus/download/latest`

## 写作主链

Flow 长篇任务默认进入 Story System：

1. 作品合同：题材、读者承诺、禁区、规则。
2. 写作任务书：章节目标、必达节拍、人物状态、伏笔与结尾落点。
3. 初稿：主笔生成正文，不把策略当成最终结果。
4. 多维审查：设定、连续性、角色一致性、逻辑、节奏、AI 味。
5. 二稿：非阻断问题自动修订，阻断问题不提交长期记忆。
6. DocumentPatch：Auto 模式可写入，审阅模式必须用户确认。
7. 章节提交：通过后抽取事件、人物状态、开放伏笔、读者承诺和章节摘要。

## 题材包

网文/文学题材包不少于 18 个：修仙、玄幻、高武、都市异能、都市日常、历史、历史脑洞、古言、宫斗宅斗、现言、狗血言情、悬疑、规则怪谈、科幻、末世、无限流、现实题材、职场婚恋。

学生写作素材包：

- 中学记叙文：成长瞬间、亲情友情、校园生活、挫折反思、细节描写、环境烘托、首尾照应。
- 中学说明文：对象特征、说明顺序、说明方法、生活科技、传统文化、校园观察、语言准确性。
- 中学议论文：中心论点、分论点、事实论据、道理论据、正反对比、递进结构、结尾升华。
- 大学记叙/散文：自我认知、专业成长、城市经验、社会实践、关系变化、克制表达。
- 大学说明/科普：概念解释、案例拆解、数据意识、学术表达、跨学科材料、引用规范。
- 大学议论文/评论：问题意识、公共议题、学术观点、现实案例、反驳段、结构化论证。

## 发布流程

1. 更新 `package.json`、`src-tauri/Cargo.toml`、`src-tauri/tauri.conf.json` 版本。
2. 执行 `npm.cmd run lint`、`npm.cmd run build`、`npm.cmd run tauri:check:portable`。
3. 使用 Tauri updater 私钥构建安装器并生成 `.sig`。
4. 将安装器、便携包和 `latest.json` 上传到主站 `/www/wwwroot/scallion.uno/public/downloads/papyrus/`。
5. 验证 `https://scallion.uno/api/papyrus/update` 返回 JSON。

私钥和密码只允许存在本机或服务器环境变量中，不得入库。
